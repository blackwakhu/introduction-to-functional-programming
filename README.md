## Basics of clojure
We begin by the hello world code
```clojure 
(println "hello world")
```
### Basic math operations
This includes the following designs
1. Addition 
  ```clojure 
    (+ 1 2)
  ```
2. Subtraction
    ```clojure 
    (- 2 1)
    ```
3. Multiplication
    ```clojure 
    (* 5 2)
    ```
4. Division
    ```clojure 
    (/ 10 2)
    ```
5. Modulo
    ```clojure 
    (mod 5 3)
    ```
